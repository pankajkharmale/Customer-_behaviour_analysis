# Customer-_behaviour_analysis
A Complete Data Analytics Project – SQL + Power BI + CSV Dataset  This project analyzes customer shopping behavior using real-world-style data, SQL insights, and an interactive Power BI dashboard. It demonstrates end-to-end data analysis — from raw CSV data to SQL exploration to visual storytelling in Power BI.

#Project Overview:
The goal of this project is to analyze customer shopping patterns, revenue trends, product performance, and behavioral segments using a multi-step analytics approach.
The project covers:
Loading and exploring data in Python
Cleaning & preprocessing the dataset
Performing SQL-based business analysis
Visualizing insights using an interactive Power BI dashboard
This project is ideal for showcasing data analytics, SQL, dashboarding, and storytelling skills to recruiters and hiring managers.

#Tools & Technologies:

Python (Pandas, NumPy, Matplotlib, Seaborn) – For loading, cleaning, EDA
MySQL Server – For writing and running SQL queries
Power BI – For building the dashboard and visual report
Jupyter Notebook / VS Code – Development environment
GitHub – Version control and project documentation
